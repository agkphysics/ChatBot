This folder contains the necessary .jar files for the bot to run correctly.

The required files are:
    - jCOLIBRI2 and its dependencies
    - java-string-similarity-0.19.jar
        available at https://github.com/tdebatty/java-string-similarity/releases/download/v0.19/java-string-similarity-0.19.jar
	- Stanford CoreNLP and its dependencies
	    available at http://stanfordnlp.github.io/CoreNLP
	    Also need the GATE Twitter model (gate-EN-twitter.model), available from https://gate.ac.uk/wiki/twitter-postagger.html
