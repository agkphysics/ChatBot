This folder contains the necessary .jar files for the bot to run correctly.

The required files are:
    - All of the .jar libraries used by jCOLIBRI2
    - opennlp-tools-1.6.0.jar, available at 
        https://www.apache.org/dyn/closer.cgi/opennlp/opennlp-1.6.0/apache-opennlp-1.6.0-bin.tar.gz
    - java-string-similarity-0.19.jar, available at 
        https://github.com/tdebatty/java-string-similarity/releases/download/v0.19/java-string-similarity-0.19.jar
